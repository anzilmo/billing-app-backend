# Billing App Backend

A simple, scalable billing software backend built using FastAPI.  
Designed for small retail businesses such as grocery shops.

---

## 🚀 Features (MVP)

- Product management (CRUD)
- Barcode-based product lookup
- Invoice generation (coming soon)
- Printable invoice support (PDF)

---

## 🧠 Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
